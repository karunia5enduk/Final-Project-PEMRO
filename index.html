ElectroEvent Hub

ElectroEvent Hub adalah sistem informasi event berbasis web yang dikembangkan untuk memfasilitasi publikasi dan manajemen kegiatan Jurusan Teknik Elektro dan Teknik Informatika.

Platform ini memungkinkan mahasiswa untuk mendaftar event, mendapatkan informasi terbaru, serta memudahkan admin dalam mengelola seluruh kegiatan jurusan secara digital.

🛠️ PHP Native 🗄️ MySQL 🎨 CSS3 Pure ⚡ Vanilla JS 🌐 HTML5
👤 Tim Developer

Anggota Kelompok 5

Mata Kuliah Pemrograman Web – Semester 4 Jurusan Teknik Informatika
